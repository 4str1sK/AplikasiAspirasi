# Aplikasi Aspirasi

Aplikasi desktop berbasis Java untuk menyampaikan kritik, saran, dan aspirasi secara digital.

## Note
Project ini masih berada pada tahap awal pengembangan karena saat ini difokuskan untuk memenuhi kebutuhan tugas sekolah.
(BUGGY)

## Features
- Login siswa & admin
- Menambahkan aspirasi
- Status aspirasi
- Feedback admin
- Dynamic card UI
- Modern UI menggunakan FlatLaf

## Tech Stack
- Java JDK 21
- Apache NetBeans 21
- Java Swing
- MySQL
- MySQL Connector/J
- FlatLaf

## Database
Database menggunakan MySQL dengan relasi:
- sim_siswa
- sim_admin
- sim_kategori
- sim_input_aspirasi
- sim_aspirasi

## Libraries
- mysql-connector-j
- FlatLaf
